🚀 DeskHero - Seat Reservation App
Dit project is een herstelde versie van de DeskHero applicatie, gerepareerd met behulp van Antigravity. De app stelt gebruikers in staat om werkplekken te reserveren in een kantooromgeving.

🛠️ Installatie & Gebruik
Om deze applicatie lokaal te draaien, volg je deze stappen:

Clone de repository:

Bash
git clone https://github.com/willepe73/DeskHero.git
cd DeskHero/seatapp_fixed
Installeer de benodigdheden:

Bash
pip install -r requirements.txt
Start de applicatie:

Bash
python app.py
De app is vervolgens bereikbaar in je browser via http://127.0.0.1:5000.

🏛️ Project Filosofie
Bij het herstellen van deze code hebben we getracht de vier deugden van het Stoïcisme in acht te nemen:

Wijsheid: Het begrijpen van de fouten in de oorspronkelijke code.

Rechtvaardigheid: De code herstellen zodat deze weer doet wat hij moet doen voor de gebruiker.

Moed: Het aangaan van de strijd met Windows Path-instellingen en Git-fouten.

Matigheid: De repository schoon en overzichtelijk houden door oude mappen te verwijderen.
